Defined in header <experimental/simd>
template< class T >
struct is_abi_tag;
 (parallelism TS v2)

If T is an ABI tag in the simd_abi namespace, provides the member constant value equal true. For any other type, value is false.

T - a type to check

inline constexpr bool is_abi_tag_v = is_abi_tag<T>::value;

Example

Run this code
#include <experimental/simd>
#include <iostream>
 
int main() 
{
    namespace stdx = std::experimental;
    std::cout << std::boolalpha
              << stdx::is_abi_tag_v<stdx::simd_abi::scalar> << '\n'
              << stdx::is_abi_tag_v<int> << '\n';
}

false
